## 🚀 Installation

Chaque skill est un dossier autonome. Pour l'utiliser avec Claude :

1. **Copie le dossier** de la skill dans ton projet
2. **Lis le `SKILL.md`** à l'intérieur — c'est le fichier que Claude lit pour savoir quoi faire
3. C'est tout. Pas de setup compliqué. 🎯
