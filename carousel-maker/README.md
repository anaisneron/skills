# 🎠 Carousel Maker

Skill Claude — Carousel Maker brandé pour Instagram & LinkedIn.

Extrait automatiquement le contenu de TikTok, YouTube et articles de blogue via Apify.

## Fonctionnalités

- Génère des carrousels 1080x1350 (format 4:5 portrait)
- Applique le branding de l'entreprise (couleurs, fonts, logo)
- Extrait le contenu de vidéos et articles automatiquement
- Export en PNG prêt à publier

## Utilisation

Copie ce dossier dans ton projet et lis le `SKILL.md` pour les instructions complètes.
